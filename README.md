# iOS-OTA-Downgrader

- **Newer (but still unsupported) Windows branch, do not use**
- Windows support can be added back when new futurerestore beta builds for Windows get released. Until then, the changes needed are within this branch.

## Supported OS versions/distros
- **Windows** 7 and newer, including 8.1 and 10
